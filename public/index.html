Git Push Deploy

Push to GitHub and your app auto-deploys. Webhooks, rollback, zero downtime — all built in.

Auto SSL & Domains

Cloudflare Tunnel gives you HTTPS, custom domains, and DDoS protection — no port forwarding needed.

Admin Dashboard

How It Works

Three steps. That's it.

01

Install Pipee

Clone the repo and run setup. Takes 2 minutes on any machine — Windows, Mac, or Linux.

git clone https://github.com/Jeffrey0117/pipee && cd pipee && npm i
02

Connect your domain

Point your domain to Cloudflare, add a tunnel CNAME — instant HTTPS, no port forwarding.

03

Push and deploy

Register a project, push to GitHub, Pipee auto-builds and deploys. Rollback any time.

git push origin main  →  live in 30s
