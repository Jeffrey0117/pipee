Features

Everything you need.
Nothing you don't.

ZIP Upload Deploy

Drag and drop a ZIP file containing your static site. Deployed instantly with zero downtime.

Built-in Auth

User registration and login with JWT tokens. No external auth service needed.

Web Console

Manage all your sites from a clean browser UI. Create, deploy, and delete sites in seconds.

SPA Support

Automatic SPA fallback for single-page apps. React, Vue, Svelte — just upload and it works.

Subdomain Routing

Each site gets its own subdomain. my-blog.yourdomain.com — clean, memorable URLs.

Multi-User

Multiple users can register and manage their own sites independently. Configurable site limits.

How It Works

Three steps. That's it.

01

Clone and start

Clone the repo, install 3 dependencies, and start the server. Works on Windows, Mac, and Linux.

git clone && npm install && npm start
02

Register and create a site

Open the console, register an account, and create a site with a unique slug like "my-blog".

03

Upload and go live

Upload a ZIP file with your static site. It's live instantly at your-slug.yourdomain.com.
